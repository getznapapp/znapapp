This page demonstrates the cross-platform invite link functionality for Znap.

How it works:

  1. User clicks an invite link like /redirect?cameraId=abc123
  2. The redirect page attempts to open the app using myapp://join?cameraId=abc123
  3. If the app is installed, it opens directly to the join screen
  4. If not installed, it falls back to the web version at /join/abc123
